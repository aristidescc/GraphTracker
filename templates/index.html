Welcome to the Graph Management System

This system allows you to manage directed graphs, track visitors, and find paths between nodes.

Features:

  • Node Management - Create, update, and delete nodes in the graph
  • Edge Management - Create connections between nodes with optional weights
  • Visitor Tracking - Create visitors and move them through the graph
  • Path Finding - Find all possible paths between two nodes
  • Operation Logging - View a history of all operations performed on the system
